      47             : /* The basic idea behind autowidth:
      48             :     We figure out how high serifs are (and if they exist at all)
      49             :     And we guess at the cap height, xheight and descender and we mark off
      50             :         zones where the serifs go (caph,caph-serifsize) (xh,xh-serifsize)
      51             :         (0,serifsize), (ds,ds+serifsize). In the future we will ignore
      52             :         those zones.
      53             :     We look at each character and we get a rough idea of the right and
      54             :         left edge of that character. (we create an array with about 60
      55             :         entries in it for a capital letter, and we divide the edge into
      56             :         60 regions and figure out roughly what the leftmost(rightmost)
      57             :         point of the character is in each region).
      58             :     The for every character pair we try to figure out what the visual
      59             :         spacing between the two letters is (we subtract off the lbearing,
      60             :         rbearing so that our result is the visual spacing if the two
      61             :         characters were jammed up against each other).
      62             :         Here we look at all regions the characters have in common and
      63             :         just find the seperation between.
      64             :         Then given this array of distances we use a heuristic to define what
      65             :         the visual spacing is.
      66             :     Then we find the "average visual distance" between two characters
      67             :         (deducting the right and left bearings so it is in some sense absolute)
      68             :         And foreach character on the left side we find the average visual
      69             :         distance of all characters paired with it and deduct this from the
      70             :         more general average found above. This gives us the amount by which
      71             :         this character's rbearing should differ from the general rbearing.
      72             :         The general rbearing should be 2*(spacing-average)/3, so the specific
      73             :         is 2*(spacing-average)/3 + average-local_average.
      74             :         Do the same thing to find lbearings for characters on the right side.
      75             : ===================================
      76             : Autokern has similar ideas, but is simpler:
      77             :     Once we've found the average of the zone distances
      78             :     Adjust the kerning between the two characters so that
      79             :         <kern> = <dspace>-(<rb>-<lb>-<visual>)
      80             :         Again check for overlap (<dspace>/4 in any zone)
      81             :     Kerning by the left character can always propigate to dependents
      82             :     Kerning by the right char should not
      83             :         (ie. A and À kern the same with V
      84             :          but V kerns differently with e and é-- the accent gets in the way)
      85             :     No, I think it is better not to propigate kerning.
      86             : */
